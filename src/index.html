    Outlook Add-in

    Outlook Add-in

    Capture + send

    Install the Apolla add-in in Outlook to capture emails and send Threshold's staged replies, drafts, and meeting invites. The manifest carries your connection, so the add-in arrives already configured — no token to paste.

    Add-in manifest downloaded to your Downloads folder. In Outlook:
    1. Get Add-ins → My add-ins → Add a custom add-in → Add from File… — on Mac “Add from URL” is greyed out, so use the downloaded file.
    2. Pick the manifest from your Downloads. It installs already configured for this workspace — no token to paste.
    Then open any email and click Threshold to bring your staged drafts forward. Already have an older “Apolla Capture” add-in installed? Remove it first (its “…” menu → Remove).
